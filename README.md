# linkcheck-action

[![License](https://img.shields.io/badge/License-Apache%202.0-blue.svg)](https://opensource.org/licenses/Apache-2.0)

A reusable GitHub Action for checking broken links on websites using [muffet](https://github.com/raviqqe/muffet).

## Status

Under development.

## About

This action provides a simple way to validate links on your website as part of your CI/CD pipeline. It uses muffet, a fast website link checker written in Go, to recursively scan pages and report broken links.
